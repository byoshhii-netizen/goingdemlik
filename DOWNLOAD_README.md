# Demlik Mağaza Projesi

## Çalıştırma

1. Node.js 18 veya üzeri kurulu olsun.
2. PostgreSQL bağlantısı için `DATABASE_URL` ortam değişkenini ayarlayın.
3. Proje klasöründe şu komutları çalıştırın:

```bash
npm install
npm start
```

Uygulama varsayılan olarak `PORT` değişkenini, yoksa 3000 portunu kullanır.

## Shopier

Shopier bilgileri uygulama içindeki admin panelinden girilir:

- `/admin.html`
- **Mağaza & Ödemeler**

PAT ve webhook token değerlerini kaynak koduna eklemeyin.
